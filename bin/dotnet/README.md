## How to Update jira v8 example

```
Epicmorg.DockerGenerator.exe
    --workdir "<path>\docker-scripts\linux\atlassian\"
    --json "<path>\atlassian-json\json-backups\archived\jira-software.json" \\
    --product "jira"
	--ignore-versions-without-templates
```
